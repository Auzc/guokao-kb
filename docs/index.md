## 🔄 迭代规则

每篇知识文档遵循以下规范：

1. **frontmatter 元数据**：包含 `title`、`tags`、`date`、`iteration`、`status`、`category`
2. **内容结构**：核心要点 → 详细内容 → B站视频资源 → 练习记录 → 迭代备注
3. **落盘流程**：每次调研迭代生成/更新文档 → Git commit → 自动部署到网站

## 🔍 如何使用

- **浏览**：左侧导航按模块分类，支持展开/折叠
- **编辑**：点击每页右上角 ✏️ 图标可直接在 GitHub 上编辑
