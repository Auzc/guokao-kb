# 备考规划总览

> 目标：建立“信息-基础-强化-冲刺”四阶段闭环，形成稳定可执行计划。

## 考试科目

| 类别 | 科目 | 说明 |
|------|------|------|
| A类（综合管理） | 行测 + 申论 | 大部分中央机关岗位 |
| B类（行政执法） | 行测 + 申论（执法卷） | 部分执法类岗位 |

> 具体以当年公告为准，参考国家公务员局官网。

## 阶段目标（简版）

- **信息与选岗**：确认可报岗位与竞争区间
- **打基础**：建立知识地图，形成每日学习节奏
- **强化真题**：把方法转化为做题速度和正确率
- **冲刺模考**：优化时间分配，压低失误率

## 执行原则

1. 先求稳，再求快
2. 每次练习都要复盘，不做“只刷不总结”
3. 以真题为核心素材，避免偏难怪题干扰
4. 行测优先资料分析，申论优先归纳概括

## 每日备考模板

=== "工作日（2-3小时）"

    | 时间 | 内容 |
    |------|------|
    | 45min | 行测模块专项（轮换） |
    | 60min | 申论练习（题型单练或材料拆解） |
    | 30min | 错题复盘 |
    | 15-30min | 时政/常识积累 |

=== "周末（4-6小时）"

    | 时间 | 内容 |
    |------|------|
    | 2-3h | 1套行测真题/模考 + 复盘 |
    | 2-3h | 1套申论套卷（或大作文+贯彻执行精批） |
