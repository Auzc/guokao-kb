# 大作文

> 申论分值最高的题型（35-40分）。核心：论点来自材料、论据结合材料与政策、结构稳定。

## 通用结构

```
第一段：引入主题 + 亮明中心论点
第二段：分论点1 + 论据（材料案例/政策/数据）
第三段：分论点2 + 论据
第四段：分论点3 + 论据
第五段：总结升华 + 展望
```
