# 提出对策

> 核心：准确定位问题 → 提出针对性、可操作、可落地的对策。

## 对策来源

- **直接对策**：材料中明确给出的建议
- **间接对策**：从问题反推（问题→对策）
- **经验借鉴**：从材料中的成功案例提炼
