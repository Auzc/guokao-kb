# 综合分析

> 常见类型：词句理解题、评论分析题、启示题、比较分析题。

## 作答方法（以词句理解题为例）

1. **解释含义**：先解释这句话/这个词在材料中的含义
2. **多角度分析**：从原因、表现、影响等角度展开
3. **总结/对策**：回扣主题，提出对策或展望
