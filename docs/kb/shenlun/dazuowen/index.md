# 大作文

> 分值最高题型（35-40 分）。核心原则：**论点来自材料、论据结合材料与政策、结构稳定、语言规范**。

## 审题要点

1. **主题词**：题目给定的关键词/句子是什么？
2. **写作身份**：政府工作人员/调研者/青年干部等
3. **文体要求**：议论文为主，不写抒情散文
4. **限制条件**：是否要求结合材料、是否给出角度

## 推荐结构（稳分结构）

| 段落 | 内容 | 目标 |
|------|------|------|
| 第一段 | 引入主题 + 亮明中心论点 | 快速破题 |
| 第二段 | 分论点1 + 材料案例/政策例证 | 论证充分 |
| 第三段 | 分论点2 + 数据/对比/制度解释 | 论证多样 |
| 第四段 | 分论点3 + 实践案例/治理路径 | 落地可行 |
| 第五段 | 总结升华 + 展望 | 收束有力 |

## 素材来源优先级

- **第一优先**：题目材料中的有效表述（优先“抄”成规范句）
- **第二优先**：政策语言/治理表述（公共服务、基层治理、创新驱动等）
- **第三优先**：通用事实素材（仅做补充，不喧宾夺主）

## 主流通关建议（基于样本库）

- 大作文应在“归纳概括+贯彻执行”稳定后再集中突破，避免空写。
- 先练“段落论证能力”，再练“完整文章成文”，最后进入限时写作。
- 推荐组合：系统课学框架 + 真题课看范文拆解 + 模考练速度。

## 样本支持度

- 课程样本：bilibili-gongkao-xiaoma-ge-shenlun、bilibili-bailu-shenlun-zhenti、bilibili-hechong-shenlun-batch。
- 路径样本：guide-common-shenlun-first、guide-mokao-weekly。
- 结论可信度：高（先题型后套卷，再限时成文）。
