# 提出对策

> 核心：准确定位问题 → 提出**针对性、可操作、可落地**的对策。

## 对策来源（三渠道）

1. **直接对策**：材料中明确给出的建议/政策/做法
2. **间接对策**：从问题反推（问题 → 原因 → 对策）
3. **经验借鉴**：从材料中的成功案例提炼可复制路径

## 结构化写法（推荐）

| 层次 | 内容 | 示例句式 |
|------|------|----------|
| 问题界定 | 精准概括问题 | “当前主要问题在于……” |
| 对策主体 | 谁来做 | “由主管部门牵头、属地落实……” |
| 对策动作 | 做什么 | “建立……机制、开展……行动……” |
| 预期效果 | 达成目标 | “从而提升……、降低……” |

## 可操作性检查清单

- 是否有明确主体（谁来干）
- 是否有动作路径（怎么干）
- 是否可衡量（如何评估效果）
- 是否符合政策语境（合法合规、治理导向）

## 主流通关建议（基于样本库）

- 对策题与归纳概括高度相关，先练好“问题提炼”会更稳。
- 先练材料导向对策，再补充政策表达提升语言质量。
- 练习顺序建议：单题型 → 真题拆解 → 套卷整合。

## 样本支持度

- 课程样本：bilibili-gongkao-xiaoma-ge-shenlun、bilibili-bailu-shenlun-zhenti。
- 路径样本：guide-common-shenlun-first。
- 结论可信度：高（先归纳后对策是常见训练顺序）。
