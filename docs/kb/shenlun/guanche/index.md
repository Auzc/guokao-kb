# 贯彻执行（公文写作）

> 分值常在 15-20 分，关键是“格式正确 + 内容完整 + 语言得体”。

## 高频文种清单

- 倡议书、建议书、讲话稿、发言稿
- 调研报告、工作方案、推荐材料、宣传稿
- 短评、简报、导言、编者按

## 结构模板（通用）

| 部分 | 内容 | 注意 |
|------|------|------|
| 标题 | 文种+事由 | 简洁明确 |
| 称谓/主送 | 对象清楚 | 按题目要求写 |
| 正文 | 背景→主体内容→结语 | 逻辑清楚、要点完整 |
| 落款/日期 | 单位与日期 | 视文种决定是否写 |

## 常见丢分点

1. **格式丢分**：漏称谓、漏落款、文种不符
2. **内容丢分**：只写原因不写对策，或层次混乱
3. **语言丢分**：口语化过重、语气不当（倡议/讲话稿尤其注意）

## 主流通关建议（基于样本库）

- 贯彻执行是“归纳+表达”的综合题，先练要点提炼会更稳。
- 建议做“文种模板卡”，考前快速复盘格式。
- 真题讲解课重点看“材料→成文”的组织逻辑。
