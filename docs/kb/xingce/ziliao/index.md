# 资料分析

> 资料分析是行测中**提分最快**的模块，目标正确率 85%+（17/20 题）。

## 模块特点

| 维度 | 说明 |
|------|------|
| 题量 | 20 题（4 篇材料，每篇 5 题） |
| 分值 | 约 20 分（每题 1 分） |
| 建议用时 | 25-30 分钟 |
| 提分难度 | ⭐⭐ 低（方法+练习即可） |
| 目标正确率 | 85%+（17 题以上） |

## 核心能力

1. **阅读能力**：快速定位数据、理解材料结构
2. **速算能力**：截位直除、特征数字法、错位加减
3. **公式应用**：增长率、增长量、比重、倍数、平均数
