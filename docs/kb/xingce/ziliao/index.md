# 资料分析

!!! success "提分性价比最高的模块"
    资料分析是行测中**最容易通过训练提分**的模块。掌握公式+速算技巧后，
    正确率可以稳定在 85%-95%。建议每天至少练 1-2 篇。

## 核心考点

- 基期/现期/增长量/增长率
- 比重（现期比重、基期比重、比重变化）
- 倍数与翻番
- 平均数（现期平均、基期平均、平均数增长率）
- 年均增长率
- 间隔增长率
