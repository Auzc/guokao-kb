# 公式体系

> 资料分析围绕五大核心概念出题，掌握公式是解题基础。

## 五大核心概念

### 1. 增长率（增长百分之几）

**公式**：
$$增长率 = \frac{现期量 - 基期量}{基期量} = \frac{增长量}{基期量}$$

**变形**：
- 已知现期、基期：$r = \frac{现期 - 基期}{基期}$
- 已知现期、增长量：$r = \frac{增长量}{现期 - 增长量}$
- 已知基期、增长量：$r = \frac{增长量}{基期}$

**速算**：增长率接近特殊值时用特征数字法（如 14.3%≈1/7）

### 2. 增长量（增长了多少）

**公式**：
$$增长量 = 现期量 - 基期量 = 基期量 \times 增长率$$

**变形**（已知现期和增长率）：
$$增长量 = \frac{现期量 \times 增长率}{1 + 增长率}$$

**速算技巧**：
- 增长率 = 1/7 时，增长量 = 现期量 / 8
- 增长率 = 1/8 时，增长量 = 现期量 / 9
- 增长率 = 1/9 时，增长量 = 现期量 / 10

### 3. 比重（占百分之几）

**公式**：
$$比重 = \frac{部分}{整体}$$

**比重变化**：
$$比重变化 = \frac{部分增长率 - 整体增长率}{1 + 部分增长率} \times 基期比重$$

**判断口诀**：
- 部分增长率 > 整体增长率 → 比重上升
- 部分增长率 < 整体增长率 → 比重下降
- 部分增长率 = 整体增长率 → 比重不变

### 4. 倍数（是几倍）

**公式**：
$$倍数 = \frac{A}{B}$$

**注意**：
- "A 是 B 的几倍" = A/B
- "A 比 B 多几倍" = (A-B)/B = A/B - 1
- "增长了几倍" = 增长率

### 5. 平均数

**公式**：
$$平均数 = \frac{总量}{份数}$$

**平均数增长率**：
$$r_{平均} = \frac{r_{总量} - r_{份数}}{1 + r_{份数}}$$

## 公式速查表

| 概念 | 核心公式 | 常见变形 |
|------|----------|----------|
| 增长率 | (现期-基期)/基期 | 增长量/(现期-增长量) |
| 增长量 | 现期-基期 | 现期×r/(1+r) |
| 比重 | 部分/整体 | 比重变化公式 |
| 倍数 | A/B | A/B - 1（多几倍） |
| 平均数 | 总量/份数 | 平均数增长率公式 |

## 常见题型

### 基期量计算

**公式**：
$$基期量 = \frac{现期量}{1 + 增长率}$$

**速算**：增长率较小时，$基期量 \approx 现期量 \times (1 - 增长率)$

### 间隔增长率

**公式**：
$$r_{间隔} = r_1 + r_2 + r_1 \times r_2$$

**适用**：求两年间的总增长率。

### 年均增长率

**公式**：
$$末期量 = 初期量 \times (1 + r)^n$$

**速算**：n 较小时用二项式展开近似。
