# 片段阅读

> 片段阅读考查阅读理解能力，需要快速抓住文段主旨。

## 题型分类

### 1. 主旨概括题

**问法**：这段文字主要说明了什么？/ 意在说明？

**解题方法**：

#### 找主旨句

| 结构类型 | 主旨句位置 | 标志词 |
|----------|------------|--------|
| 总分结构 | 首句 | 总之、因此、所以 |
| 分总结构 | 尾句 | 总之、因此、所以 |
| 总分总结构 | 首尾句 | 总之、因此、所以 |
| 并列结构 | 概括全段 | 同时、此外、另外 |
| 转折结构 | 转折后 | 但是、然而、却 |

**解题步骤**：
1. **看首尾句**：通常主旨句在首句或尾句
2. **找标志词**：转折、因果、总结词
3. **概括主旨**：用自己的话复述
4. **匹配选项**：选择最接近的选项

### 2. 意图判断题

**问法**：这段文字意在强调？/ 作者的意图是？

**与主旨题的区别**：
- 主旨题：文段说了什么
- 意图题：作者想说什么（可能有引申）

**解题方法**：
1. **先找主旨**：和主旨题一样
2. **判断是否需要引申**：
   - 问题类文段 → 选对策
   - 观点类文段 → 选观点

### 3. 细节判断题

**问法**：以下说法正确/错误的是？

**解题方法**：
1. **看选项**：标记关键词
2. **定位原文**：找对应语句
3. **对比判断**：选项与原文是否一致

**常见错误选项**：
- 偷换概念：概念被替换
- 偷换时态：将来变已经
- 偷换范围：部分变全部
- 无中生有：原文没提到

### 4. 词句理解题

**问法**：文中"xxx"指的是？

**解题方法**：
1. **定位词句**：找到在文中的位置
2. **看上下文**：理解语境
3. **代入选项**：看哪个最通顺

## 解题技巧

### 快速阅读法

1. **看首尾句**：快速把握主旨
2. **找标志词**：转折、因果、总结
3. **跳过举例**：例子通常不是重点
4. **关注结论**：结论通常是主旨

### 选项排除法

1. **排除绝对化**：一定、必须、所有（通常错误）
2. **排除无中生有**：原文没提到
3. **排除偷换概念**：概念被替换
