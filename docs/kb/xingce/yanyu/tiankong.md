# 逻辑填空

> 逻辑填空考查词语辨析和语境分析能力，需要大量积累。

## 解题方法

### 方法 1：语境分析

**适用**：通过上下文推断空缺处含义。

**步骤**：
1. **找提示词**：空缺处前后的关键词
2. **判断关系**：解释、并列、转折、递进
3. **推断含义**：根据关系推断空缺处意思

**常见关系**：

| 关系类型 | 标志词 | 解题思路 |
|----------|--------|----------|
| 解释关系 | 也就是说、即、换言之 | 找同义词 |
| 并列关系 | 和、与、同时、也 | 找近义词 |
| 转折关系 | 但是、然而、却 | 找反义词 |
| 递进关系 | 更、甚至、不仅...而且 | 找程度更深的词 |

### 方法 2：词语辨析

**适用**：选项词语意思相近时。

**辨析维度**：

| 维度 | 说明 | 示例 |
|------|------|------|
| 含义侧重 | 强调的重点不同 | "遏止"侧重停止，"遏制"侧重控制 |
| 语义轻重 | 程度不同 | "批评"轻，"批判"重 |
| 感情色彩 | 褒贬中性 | "果断"褒，"武断"贬 |
| 搭配对象 | 固定搭配 | "发扬"搭配精神，"发挥"搭配作用 |
| 语体风格 | 书面/口语 | "父亲"书面，"爸爸"口语 |

### 方法 3：固定搭配

**适用**：考查常见词语搭配。

**积累方法**：
1. **读人民日报**：培养语感
2. **背高频搭配**：整理常见搭配
3. **做真题积累**：记录错题中的搭配

## 常见错误

| 错误类型 | 原因 | 改进 |
|----------|------|------|
| 凭语感选 | 没有分析语境 | 先分析语境再选 |
| 忽略搭配 | 不了解固定搭配 | 积累常见搭配 |
| 辨析不清 | 近义词区分不开 | 从含义侧重入手 |
