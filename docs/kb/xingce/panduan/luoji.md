# 逻辑判断

> 逻辑判断是判断推理中难度最大的题型，但掌握方法后可稳定拿分。

## 两大题型

### 1. 必然性推理（形式逻辑）

**特征**：给定条件，推出必然结论。

**核心知识点**：

#### 直言命题

| 命题类型 | 形式 | 矛盾命题 |
|----------|------|----------|
| 所有是 | 所有 A 是 B | 有些 A 不是 B |
| 所有非 | 所有 A 不是 B | 有些 A 是 B |
| 有些是 | 有些 A 是 B | 所有 A 不是 B |
| 有些非 | 有些 A 不是 B | 所有 A 是 B |

**推理规则**：
- "所有是" → "有些是"（下真上必真）
- "所有非" → "有些非"（下真上必真）

#### 假言命题

**形式**：如果 A，那么 B（A→B）

**推理规则**：
- **肯前肯后**：A 真 → B 真
- **否后否前**：B 假 → A 假
- **否前未必否后**：A 假 → B 不确定
- **肯后未必肯前**：B 真 → A 不确定

**矛盾命题**：A 且非 B

#### 联言命题与选言命题

| 命题类型 | 形式 | 真假判断 |
|----------|------|----------|
| 联言命题 | A 且 B | 全真才真，一假即假 |
| 相容选言 | A 或 B | 一真即真，全假才假 |
| 不相容选言 | 要么 A 要么 B | 只能一真 |

### 2. 可能性推理（加强/削弱）

**特征**：给定论据和结论，判断选项对结论的影响。

**常见问法**：
- 以下哪项最能削弱上述论证？
- 以下哪项最能加强上述论证？
- 以下哪项是上述论证的前提假设？

**解题步骤**：
1. **找结论**：结论通常在"因此"、"所以"、"可见"之后
2. **找论据**：支持结论的理由
3. **分析逻辑漏洞**：论据和结论之间的跳跃
4. **判断选项**：加强/削弱/无关

**加强方式**：
- 补充论据（直接支持结论）
- 建立联系（论据→结论的桥梁）
- 排除他因（排除其他可能性）

**削弱方式**：
- 否定论据（论据不成立）
- 切断联系（论据→结论的桥梁断裂）
- 另有他因（存在其他解释）

## 解题技巧

### 必然性推理

1. **画表格**：复杂条件用表格整理
2. **找突破口**：从确定信息入手
3. **假设法**：不确定时假设验证

### 可能性推理

1. **紧扣结论**：选项必须与结论相关
2. **力度比较**：直接 > 间接，必然 > 可能
3. **排除无关**：与结论无关的选项直接排除

## 练习建议

1. **先学形式逻辑**：掌握命题和推理规则
2. **再学可能性推理**：加强/削弱/前提假设
3. **混合练习**：识别题型，选择方法
