# 类比推理

> 类比推理是判断推理中最简单的题型，目标正确率 90%+。

## 常见关系类型

### 1. 概念关系

| 关系类型 | 示例 | 说明 |
|----------|------|------|
| 全同关系 | 番茄:西红柿 | 同一事物不同叫法 |
| 包含关系 | 水果:苹果 | 大类包含小类 |
| 交叉关系 | 学生:党员 | 有交集但不完全重合 |
| 全异关系 | 生:死 | 完全对立 |
| 并列关系 | 苹果:香蕉 | 同属一类但互不包含 |

### 2. 逻辑关系

| 关系类型 | 示例 | 说明 |
|----------|------|------|
| 因果关系 | 下雨:地湿 | 前因后果 |
| 条件关系 | 努力:成功 | 前者是后者的条件 |
| 目的关系 | 跑步:减肥 | 前者是手段，后者是目的 |
| 转折关系 | 虽然:但是 | 逻辑转折 |

### 3. 语法关系

| 关系类型 | 示例 | 说明 |
|----------|------|------|
| 主谓关系 | 鸟:飞 | 主语+谓语 |
| 动宾关系 | 吃:饭 | 动词+宾语 |
| 偏正关系 | 红色:苹果 | 修饰语+中心词 |
| 并列关系 | 山:水 | 词语并列 |

### 4. 语义关系

| 关系类型 | 示例 | 说明 |
|----------|------|------|
| 近义关系 | 美丽:漂亮 | 意思相近 |
| 反义关系 | 高:低 | 意思相反 |
| 象征关系 | 鸽子:和平 | 事物象征意义 |

### 5. 常识关系

| 关系类型 | 示例 | 说明 |
|----------|------|------|
| 功能关系 | 筷子:吃饭 | 事物用途 |
| 材料关系 | 木头:桌子 | 制作材料 |
| 场所关系 | 医生:医院 | 工作场所 |
| 工具关系 | 司机:汽车 | 使用工具 |

## 解题步骤

1. **看题干**：分析词语间关系
2. **看选项**：找关系相同的选项
3. **验证**：确认关系一致

## 常见错误

| 错误类型 | 原因 | 改进 |
|----------|------|------|
| 关系判断错误 | 没有分析清楚 | 先确定关系再看选项 |
| 忽略顺序 | 词语顺序颠倒 | 注意前后顺序 |
| 过度联想 | 想得太复杂 | 选择最直接的关系 |
