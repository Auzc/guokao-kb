# 真题库

> 真题是检验学习效果的唯一标准。核心原则：**先限时做，再精讲复盘**。

## 执行流程

1. **限时作答**：模拟真实考试节奏
2. **对答案标记**：区分“会而错”与“不会”
3. **逐题复盘**：还原正确解题路径
4. **错因归类**：计算错误/方法错误/信息定位错误
5. **针对性补强**：回知识库对应模块补短板

## 年份优先级（基于样本库共识）

| 优先级 | 年份 | 说明 |
|--------|------|------|
| 最高 | 2026国考 | 最新真题，最贴近考试风格 |
| 最高 | 2025国考 | 近两年真题必做 |
| 高 | 2024国考 | 近三年覆盖 |
| 中 | 2023国考 | 查漏补缺 |
| 中 | 2022国考 | 基础巩固 |

## 样本支持度

- 路径样本：guide-zhenti-yearly（近5年真题优先共识）
- 结论可信度：高

## 使用建议

- 基础期：不建议整卷刷，先做分题型真题
- 强化期：近5年真题分模块拆解
- 冲刺期：限时成套做，重点练时间分配和心态管理
